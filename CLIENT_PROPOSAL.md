## The Solution: GenZ Restaurant POS

A complete, web-based restaurant management system that runs on any device (laptop, tablet, phone) through a browser.

### Core Features Included

#### 1. Point of Sale (POS)
- Table management with visual layout (10+ tables)
- Quick order taking with category-wise menu
- Split bills, merge tables, apply discounts
- Multiple payment methods (Cash, Card, UPI, Wallet)
- GST-compliant invoices with automatic tax calculation

#### 2. Kitchen Order Ticket (KOT)
- Real-time order display in kitchen
- Status tracking: Pending → Preparing → Ready → Served
- Timer display for order priority
- Multiple kitchen station support

#### 3. Menu Management
- **170+ pre-loaded menu items** (from GenZ Restaurant)
- Categories: Starters, Main Course, Breads, Biryani, Beverages, etc.
- Image support for each item
- Real-time availability toggle
- Price management with GST settings

#### 4. Inventory Management *(Optional Add-on)*
- Stock tracking with auto-deduction on sales
- Low-stock alerts
- Supplier management
- Purchase order generation

#### 5. Reports & Analytics
- Daily sales report
- Payment mode summary
- Top-selling items
- Staff performance tracking
- Export to PDF/Excel

#### 6. Customer Management *(Optional Add-on)*
- Customer database with order history
- Loyalty points system
- SMS/WhatsApp integration for marketing
